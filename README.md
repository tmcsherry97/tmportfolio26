This repository contains IT/Help Desk support labs created to demonstrate entry-level help desk skills through documentation and validation. The projects focus on common Tier 1 responsibilities, including identity and access management, user onboarding, troubleshooting, and ticketing workflows.

---

## Projects

### Microsoft 365 & Microsoft Entra ID Lab
Configured an instance of Microsoft 365 for Business and Microsoft Entra ID to simulate user onboarding, security group management, and principle of least privilege through role-based access control. Implemented multi-factor authentication and deployed a shared Outlook mailbox and SharePoint site with appropriate permissions. Access and functionality were validated using a standard end-user account to confirm expected behavior and troubleshooting steps.

[**Full PDF Documentation w/ images**](./pdfs/ms365entralab.pdf)

---

### Active Directory & Group Policy Lab
Configured a Windows Server 2022 Active Directory environment within a virtualized network. Created users, groups, and organizational units to reflect standard business practices, and configured Group Policy Objects to reinforce identity and access management principles. Simulated common help desk tasks including password resets and network drive mapping.

[**Full PDF Documentation w/ images**](./pdfs/adlab.pdf)

---

### Ticketing System Workflow & Documentation Lab
Deployed and tested a Spiceworks SaaS ticketing platform to simulate end-user ticket submission and help desk workflow. Documented ticket intake, communication gaps, and resolution notes, and created a tutorial-style knowledge base article to demonstrate clear and effective documentation practices.
